
# Weather_app_Backend

You can find weather for any place by searching here.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Arindam01gi/Weather_app_Backend.git
```

Go to the project directory

```bash
  cd Weather_app_Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
